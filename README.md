killbill-oss-parent
===================

Kill Bill OSS Parent: base pom for the various Kill Bill projects.

Kill Bill compatibility
-----------------------

| OSS parent version | Kill Bill version |
| -----------------: | ----------------: |
| 0.94.y             | 0.16.z            |
| 0.140.y            | 0.18.z            |

Installation
------------

To install the pom.xml in your local repository, use maven:

    mvn clean install
